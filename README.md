# GitHub Dependencies Next


## Development:

> Requires: Docker, Node.js (>=18)

* Build Images:
  * ```shell
    $ make build-development  
    ```
* Start Containers:
  * ```shell
    $ make start-development  
    ```
* Stop Containers:
  * ```shell
    $ make stop-development  
    ```
* Remove Containers:
  * ```shell
    $ make remove-development  
    ```
